# DASAN 민원 데이터 공개 시스템 PRD

## 1. 개요
본 문서는 DASAN 민원 데이터 공개 시스템의 기능 요구사항(Product Requirements)을 정리한 문서이다.  
실시간 민원 키워드, 지역별·기간별 분석, 시각화 및 다운로드 기능을 제공한다.

---

## 2. 기능 요구사항 (Functional Requirements)

| REQ No. | 요구기능 | LEVEL 1 | LEVEL 2 | 내용 |
|--------|----------|---------|---------|------|
| SFR-003 | 전체 민원 데이터 건수 안내 | MAIN | 데이터 현황 | 데이터 수집 기간, 최신 업데이트 시각, 데이터 크기 등을 대시보드 형태로 제공 |
| SFR-004 | 키워드 기준 민원 데이터 조회 | MAIN | 데이터 검색 | 검색 결과를 리스트·차트 형태로 제공하며, 검색 키워드와 관련된 최신 발생 건수 표시 |
| SFR-005 | 특정 키워드 최신 발생 추이 | MAIN | 최신 다빈도 | 최근 일정 기간(예: 24시간, 1주일) 동안의 발생 추이를 시계열로 제공 |
| SFR-006 | 키워드 기준 민원 지도 시각화 | 지역별 | 민원 지도 | 키워드로 검색된 민원 건수를 지도에 표시하고, 건수 및 주요 속성을 툴팁으로 제공 |
| SFR-007 | 자치구·행정동 단위 분석 | 지역별 | 민원 지도 | 자치구별 발생 건수 집계 및 행정동 단위 패턴 분석 |
| SFR-008 | 핵심 키워드 요약 및 추이 | 기간별 | 키워드 분석 | 키워드별 일간 발생량 시계열 그래프 제공, 특정 날짜 클릭 시 상세 건수 확인 |
| SFR-009 | 민원 발생 빈도 분석 | 기간별 | 계절 분석 | 월별 증감 추세, 요일·시간대 분포, 평일/주말 및 계절별 패턴 분석 |
| SFR-010 | 데이터 다운로드 | 공통 | 다운로드 | CSV, Excel, 그래프 이미지 다운로드 지원 |
| SFR-011 | 연관 키워드 워드클라우드 | 공통 | 워드클라우드 | 검색 키워드와 연관성이 높은 단어를 빈도 기반으로 추출하여 워드클라우드 제공 |
| SFR-012 | 키워드 연관어 분석 | 공통 | 연관어 분석 | 키워드 간 관계를 시각적으로 표현하여 민원 패턴 확인 |
| ECR-005 | 관리자 로그 조회 | 관리자 | 로그 조회 | 화면 조회수, 이용자 수, 메뉴별 클릭 수 및 다운로드 로그 기록 |
| SFR-001 | 메뉴별 활용 통계 | 관리자 | 통계 | 메뉴별 사용 통계 제공 |

---

## 3. 비고
- REQ No 중복(SFR-006) 문제를 해소하고 기능 단위로 재정의함
- LEVEL 1 / LEVEL 2 체계를 일관되게 정리함
- 본 문서는 화면·DB 설계와 분리된 요구사항 기준 문서(PRD)로 활용
