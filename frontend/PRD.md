# 요구사항 정의서 (PRD)

# 개요 및 목표

다산콜재단120 민원공개 시스템의  Html 및 자바스크립을 이용하여 프로토타입 화면을 생성 하려고 한다.

- **프로젝트명**: 장비톡 플랫폼
- **목표**: 건설 현장에서 중장비가 필요한 발주자와 인근의 장비 공급자를 실시간으로 연결
- **핵심 가치**: 근거리 신속한 매칭, 투명한 거래 과정, 사용자 편의성 극대화
- **플랫폼 범위**:
  - 사용자 앱: Android, iOS (React Native with Expo)
  - 관리자 콘솔: 웹 기반 (Next.js + MUI)
- **참조 문서**: CLAUDE.md (기술 스택), Uiux.md (디자인), Equipment Model Options.md (장비 옵션)

# 주요 화면 기능

## 메인화면
- **데이터 현황** 전체 민원 데이터의 건수를 기간별·유형별로 안내
   ①데이터 수집 기간, 최신 업데이트 시각, 데이터 크기 등을 대시보드 형식
- **데이터 검색** 키워드를 기준으로 관련 민원 데이터를 조회
   ①검색 결과는 리스트·차트 형태로 출력
   ②검색 키워드와 관련된 최신 발생 건수
- **최신 다빈도** 특정 키워드 검색 시, 키워드와 관련된 민원이 최신으로 얼마나 발생는지 
   ①최근 일정 기간(예: 24시간, 1주일) 동안 발생 추세

### 민원 분류 및 키워드
교통·주차, 도로·시설물, 환경·청소, 건축·주택, 복지·보건, 세무·요금, 안전·재난, 문화·관광, 교육·청소년, 민원행정, 기타

## 지역별 화면
- **민원지도** 키워드로 검색된 민원 건수를 네이버 지도에 표시
  ①결합 결과를 네이버 지도 상에 표시
  ②민원 건수 및 주요 속성을 툴팁표시
- 자치구 및 행정동 단위 분석
  ①각 자치구별 발생 건수를 집계
  ②행정동 집계 자치구별 패턴 분석

## 기간별 화면
- **키워드분석** 키워드로 검색된 민원 건수를 네이버 지도 위에 표시
   ①키워드별 일간 발생량을 시계열 그래프
   ②특정 날짜 클릭 시 상세 건수 확인


- **계절 분석** 민원데이터 발생빈도
   ①월별 발생량을 추적하여, 증가·감소 추세를 파악
   ②요일 및 시간대별로 민원 건수 분포
   ③평일과 주말을 구분하여 패턴을 분석
   ④계절(봄·여름·가을·겨울)을 구분하여 패턴을 분석

# 각 화면의 공통 기능
- **다운로드** 데이터 다운로드
   ①CSV, Excel, 그래프 이미지 
- **워드클라우드** 해당 키워드와 연관성이 높은 단어를 빈도 기반으로 추출
  ①빈도 기반으로 추출해 워드클라우드
- **연관어 분석** 키워드 간 관계 시각적으로 표현
  ①민원 패턴을 확인


# 관리자 화면
- **로그조회** 화면 조회수, 이용자 수 등 확인
  ①화면 조회수, 이용자 수 등 확인
  ②메뉴별 클릭 수
  ③다운로드 로그 기록

- **통계** 메뉴별 활용 통계 기능
  ①화면 조회수, 이용자 수 등 확인
  ②메뉴별 클릭 수
  ③다운로드 로그 기록
