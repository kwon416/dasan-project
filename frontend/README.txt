================================================================================
다산콜재단 120 민원공개시스템 - 프로토타입
================================================================================

1. 프로젝트 개요
--------------------------------------------------------------------------------
서울시 120다산콜재단 민원공개시스템의 프로토타입 화면입니다.
HTML, CSS, JavaScript로 구현된 정적 프로토타입으로, 실제 기능은 없습니다.

2. 실행 방법
--------------------------------------------------------------------------------
방법 1: 파일 직접 열기
  - index.html 파일을 더블클릭하여 브라우저에서 열기
  - 또는 main1.html 파일을 직접 열기

방법 2: VS Code Live Server 사용 (권장)
  1. VS Code에서 프로젝트 폴더 열기
  2. Live Server 확장 설치 (이미 설치된 경우 생략)
  3. index.html 우클릭 > "Open with Live Server" 선택

방법 3: Python 간이 서버
  1. 터미널에서 프로젝트 폴더로 이동
  2. python -m http.server 8000 실행
  3. 브라우저에서 http://localhost:8000 접속

3. 파일 구조
--------------------------------------------------------------------------------
dasan120/
├── index.html              # 시작 페이지 (main1.html로 리다이렉트)
├── main1.html              # 메인 대시보드 (Home)
├── main2.html              # 지역별 조회 (Leaflet 지도)

├── admin.html              # 관리자 대시보드 (메뉴에서 제외됨)
├── public/
│   ├── logo/
│   │   └── logo.jpg        # 로고 이미지
│   └── images/
│       └── seoul-map.png   # 서울 지도 이미지
├── PRD.md                  # 요구사항 정의서

4. 화면 설명
--------------------------------------------------------------------------------
[main1.html - 메인 대시보드]
  - 전체 민원 데이터 현황 (기간별, 유형별)
  - 키워드 검색 기능 UI
  - 지역별 민원 지도 (Leaflet)
  - 민원 추이 분석 차트
  - 워드클라우드

[main2.html - 지역별 조회]
  - 서울시 자치구별 민원 분포 지도
  - 자치구 선택 시 상세 정보 표시
  - 키워드 검색 연동 UI

[complaint-status.html - 민원현황]
  - 민원 목록 테이블
  - 카테고리별 필터
  - 상태별 필터 (처리중, 완료 등)

[period.html - 기간별 분석]
  - 일별/주별/월별 민원 추이 그래프
  - 요일/시간대별 히트맵
  - 계절별 분석
  - 워드클라우드, 연관어 분석

[admin.html - 관리자] (네비게이션 메뉴에서 제외됨, 직접 접근만 가능)
  - 조회수, 방문자 통계
  - 메뉴별 클릭 수
  - 활동 로그
  - 다운로드 로그

* 네비게이션 메뉴: Home, 지역별 조회, 민원현황, 기간 조회 (4개)

5. 사용된 라이브러리 (CDN)
--------------------------------------------------------------------------------
- Google Fonts (Noto Sans KR)
- Material Icons
- Lucide Icons (https://unpkg.com/lucide@latest)
- Chart.js (차트)
- Leaflet (지도)

6. 민원 분류 카테고리
--------------------------------------------------------------------------------
교통/주차, 도로/시설물, 환경/청소, 건축/주택, 복지/보건,
세무/요금, 안전/재난, 문화/관광, 교육/청소년, 민원행정, 기타

7. 참고사항
--------------------------------------------------------------------------------
- 모든 데이터는 샘플 데이터입니다 (실제 데이터 아님)
- 검색, 필터, 다운로드 등 기능은 UI만 구현되어 있습니다
- 반응형 디자인 적용 (데스크톱, 태블릿, 모바일)
- UI 디자인 참조: Modernize-Nextjs-Free-main UI/ 폴더
